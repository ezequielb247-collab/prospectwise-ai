# Plano do MVP — ProspectWise AI

1. [x] Inicializar a aplicação e definir a base visual responsiva.
2. [x] Modelar campanhas, leads, análises, mensagens, conversas, CRM e preferências.
3. [x] Preparar login e isolamento por usuário para a integração de autenticação.
4. [x] Criar os fluxos de campanha, busca simulada, qualificação e mensagens.
5. [x] Adicionar fila segura simulada, opt-out, CRM e painel de métricas.
6. [x] Cobrir regras essenciais com testes e validar lint, tipos e build.
7. [x] Documentar instalação, integrações futuras e limites de segurança.
8. [ ] Publicar e conectar o repositório remoto após autorização do usuário.

## Limites desta entrega

- Nenhuma API paga será conectada.
- Nenhuma mensagem real será enviada.
- Busca, análise por IA e WhatsApp usarão adaptadores simulados substituíveis.
